### âœ… **Short Note on Loops in JavaScript**

Loops are used to **execute a block of code repeatedly** based on a condition.

---

### ğŸ” **Types of Loops:**

1. **`for` loop** â€“ Used when the number of iterations is known.

   ```javascript
   for (let i = 0; i < 5; i++) {
     console.log(i);
   }
   ```

2. **`while` loop** â€“ Repeats as long as the condition is true.

   ```javascript
   let i = 0;
   while (i < 5) {
     console.log(i);
     i++;
   }
   ```

3. **`do...while` loop** â€“ Runs **at least once**, then checks condition.

   ```javascript
   let i = 0;
   do {
     console.log(i);
     i++;
   } while (i < 5);
   ```

---

### ğŸ”„ **Special Loops for Arrays:**

* `for...of` â€“ Loops through array values

  ```javascript
  for (let color of colors) {
    console.log(color);
  }
  ```

* `for...in` â€“ Loops through object keys or array indexes

  ```javascript
  for (let index in colors) {
    console.log(colors[index]);
  }
  ```

### âœ… **Short Note on String Methods in JavaScript**

Strings in JavaScript are used to store and manipulate **text**. JavaScript provides several **built-in methods** to work with strings.

### ğŸ’¡ **Common String Methods:**

| ------------------- | ------------------------------------- | ---------------------------------------- |

### âœ… **Short Note on Conditional Statements in JavaScript**

Conditional statements are used to **make decisions** based on **conditions**. They allow your program to take different paths.

---

### ğŸ” **Types of Conditional Statements:**

1. ### **`if` Statement**

   Executes a block if the condition is true.

   ```javascript
   if (age > 18) {
     console.log("Adult");
   }
   ```

2. ### **`if...else` Statement**

   Executes one block if condition is true, another if false.

   ```javascript
   if (marks >= 40) {
     console.log("Pass");
   } else {
     console.log("Fail");
   }
   ```

3. ### **`if...else if...else`**

   Checks multiple conditions.

   ```javascript
   if (marks >= 75) {
     console.log("Distinction");
   } else if (marks >= 40) {
     console.log("Pass");
   } else {
     console.log("Fail");
   }
   ```

4. ### **`switch` Statement**

   An alternative to many `if...else` conditions.

   ```javascript
   let day = "Monday";
   switch (day) {
     case "Monday":
       console.log("Start of week");
       break;
     case "Friday":
       console.log("Weekend soon");
       break;
     default:
       console.log("Midweek");
   }
   ```
